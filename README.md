# Face AI Project

This repository is for a face recognition and facial effects AI internship project.

## Project Goal

Build a computer vision system that covers:

- Face detection
- Facial landmark detection
- Face alignment
- Face recognition
- Real-time facial effects
- Model optimization and deployment

## Week 1 Goals

- Set up the project repository
- Configure the Python development environment
- Run an OpenCV image processing demo
- Run a PyTorch environment test
- Build the first face detection baseline
